介绍当前博客的搭建流程

发表于 2025-07-07

环境准备

Hexo基于node.js，需要先安装运行环境。有两种方式安装

  • 直接官网下载node的LTS（长期支持版本）
  • 通过nvm来下载 推荐

安装Hexo

npm install -g hexo-cli
安装完成后执行hexo -v验证安装

初始化博客项目

hexo init my-blog my-blog 为项目文件夹名
cd my-blog
npm install

本地启动

推荐用VScode打开项目

  • hexo clean
  • hexo g
  • hexo server

配置博客

修改配置文件

title: 你的博客标题
subtitle: 副标题
description: 博客描述
author: 你的名字
language: zh-CN # 语言（中文）
theme: landscape # 默认主题（可更换）
url: https://你的域名.com # 如果有域名

更换主题

git clone https://github.com/next-theme/hexo-theme-next themes/next
修改 _config.yml 中的 theme 为 next

写文章

hexo new "文章标题"
会在 source/_posts 生成 .md 文件
